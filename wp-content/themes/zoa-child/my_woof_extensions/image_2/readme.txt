This image extension was created for using on the sites with WPML installed where
color terms titles+slugs needs translation to anoter language - in this case default color extension 
doesn work. Just disable default Image extension 1.0 in tab Advanced and in the same tab install image_2.zip OR upload folder image_2 into:
wp-content/my_woof_extensions + in tab Advanced -> tab Options in field 'Custom extensions folder' 
write 'my_woof_extensions'

All options for each term you will find in its options page: http://clip2net.com/s/3y5F9FG
Example: wp-admin/edit-tags.php?taxonomy=pa_size&post_type=product - click edit
